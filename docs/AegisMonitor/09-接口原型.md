# AegisMonitor 接口原型

## 1. 设计原则

接口原型用于约定 Agent、后端和前端之间的数据交互格式。后续正式编码时可根据本文件生成 Swagger/OpenAPI 文档。

接口设计原则：

- 前端只访问后端，不直接访问 Agent。
- Agent 主动向后端 HTTP 上报。
- 用户请求使用 JWT 鉴权。
- Agent 请求使用 Agent Token 或 Agent Secret 鉴权。
- 高频指标写入 InfluxDB，接口返回前端可直接绘图的数据结构。
- 时间字段统一使用 ISO-8601 字符串或毫秒时间戳，后续实现时二选一固定。

## 2. 通用约定

### 2.1 基础路径

```text
http://localhost:8080/api
```

### 2.2 用户鉴权 Header

```text
Authorization: Bearer <jwt>
```

### 2.3 Agent 鉴权 Header

注册时：

```text
X-Agent-Register-Token: <registerToken>
```

注册成功后的上报：

```text
X-Agent-Id: <agentId>
X-Agent-Secret: <agentSecret>
```

### 2.4 通用响应结构

```json
{
  "success": true,
  "code": "OK",
  "message": "success",
  "data": {}
}
```

错误响应：

```json
{
  "success": false,
  "code": "AGENT_TOKEN_INVALID",
  "message": "Agent register token is invalid",
  "data": null
}
```

## 12. 前端轮询建议

| 页面 | 接口 | 周期 |
| --- | --- | --- |
| 总览仪表盘 | 主机列表、告警事件、指标摘要 | 5 秒 |
| 主机详情 | 主机详情、最近 10 分钟指标 | 5 秒 |
| 告警中心 | 告警事件列表 | 10 秒 |
| 服务组件 | 服务列表、服务指标 | 10 秒 |
| Agent 管理 | Agent 列表 | 10 秒 |

## 13. 待详细设计确认项

- `agentSecret` 是否落库加密。
- 用户密码哈希算法，建议 BCrypt。
- InfluxDB bucket、measurement、tag 命名。
- 告警去重策略的时间窗口。
- API 错误码列表。
- 分页参数是否统一为 `page/pageSize`。
